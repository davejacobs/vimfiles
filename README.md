# Vimfiles

My Neovim configuration.

## Prerequisites

- Curl
- Git
- NeoVim (for Vim support, check out the `vim` branch)
- Nerd font for the font you're using in you're terminal

## Install

    git clone https://github.com/davejacobs/vimfiles.git
    cd vimfiles
    bin/install
